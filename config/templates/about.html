About the Project

The goal of this project is to give a larger audience access to a catalogue of classical books, available in human readable and machine readable format. All the books here are sourced from github.com/PerseusDL/canonical-latinLit.git which is a part of Open Greek and Latin Project . All the books are in TEI XML format. More about the format can be learnt here.

How to Read the Books?

Books are available to read and download in the original TEI XML and HTML format. (HTML Documents are generated with the help of teigarage.tei-c.org). There is also some integration with The Internet Archive: a book can be automatically searched from the Details tab. Most books can also be read on Scaife Viewer.

Registered users can also mark some books as favourite to return to them later.
