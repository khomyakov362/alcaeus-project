About the Project

The goal of this project is to give a larger audience access to a catalogue of classical books, available in human readable and machine readable format. All the books here are sourced from github.com/PerseusDL/canonical-latinLit.git which is a part of Open Greek and Latin Project . All the books are in TEI XML format. More about the format can be learnt here.
